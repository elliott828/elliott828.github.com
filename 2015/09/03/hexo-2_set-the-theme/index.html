Hexo博客优化

发表于 2015-09-03   |   分类于 Hexo|博客技术   |  

现在这个博客基本处于裸奔状态，除了默认的Landscape模板和仅有的两三篇文章以外，其它设置几乎是空白。对于处女座而言简直不能忍。

1. 设置Hexo博客主题

1.1 Hexo主题库

干正事儿之前，先去Hexo的主题库里溜达溜达，找一些喜欢的主题备着：

  • Alex-demo: 色彩和字体搭配都是我喜欢的。
  • Hueman-demo: 类似lofter的板式，视觉效果好，对个人博客来说略繁琐。
  • Landscape plus-demo: 对每篇文章长度无限制，限制了单页文章浏览。
  • Mabao-demo: 纯粹喜欢顶置图片。。。
  • Metro Light-demo: 灰白底色搭配有点费眼，但是整体搭配赞。
  • Modernist-demo: 最喜欢的模板，TL归档设定和标签页独立的设计都不错，不知道有没有分类归档。NexT模版也是同样效果。
  • TKL-demo: 如文档描述所说，一个设计优雅的响应式主题。

2.4 边栏

三种参数：post, always, hide
always表示在所有页面自动展开边栏；hide表示只在点击边栏按钮的时候展开；默认设置post表示在文章旁边自动展开。
